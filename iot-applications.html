Activity 1 - IoT Apps

  • What is an IoT Application?

    • IoT applications are the kind of applications that communicate with IoT devices over the internet to operate, manage and share data with them without human interference.

  • Can you think of some examples?

    • Examples :

    • self driving cars (tesla)
    • smart home set up
    • smart wearables (watches, rings etc)
    • Smart Security Systems
    • farming ( soil monitoring, irrigation systems )
    • Dizziness alert system
    • Smart Waste Management System
    • Predictive analysis for smart farming
    • AR glasses like Google glasses
    • smart ads campaign

  • What are the main features needed in an IoT application?

    • Main features:Sensors, data management, communication through web based applications/sites, security.
    More features for IoT applications include: Connectivity Management, Integration, Scope and Scalability features like automated bootstrapping, microservices architecture, Data and Hardware Security features like Crypto acceleration, random number generator.

Activity 2 - Smart Building

  • What does smart mean?

    • Integration of devices which are used to work together with the data that is being collected and stored over the network.
    • The keyword smart reflects the optimal management of collected data through AI and machine learning algorithms over a integrated network of internet of things.
    • Smart is something or device that can gather data and process it to give ample results.

  • Think of some smart building scenarios where this will be beneficial.

    • security systems
    • lightning management
    • temperature monitor(air conditioner and room heater controller)
    • hazard detection
    • Ease of Accessibility
    • Structural health monitoring

  • What are the parts needed for it to work?

    • Features: data storage, Real Time interaction 24/7, collecting data, wireless communication using wifi or bluetooth. Various parts are needed for a smart device to work:

    1. Transceiver
    2. Output devices like actuators, indicators and alarms.
    3. Sensors like temperature, humidity, light, Ventilation, Predictive maintenance.
    4. Batteries for power management.
    5. Circuit safety protocols.
